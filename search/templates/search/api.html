https://quran-search-api.herokuapp.com/api/allVerses

returns all verses

https://quran-search-api.herokuapp.com/api/search/{keywords}

takes string and then return all verses that contains this string

https://quran-search-api.herokuapp.com/api/getSurah/{surah_id}

  • takes surah number and then returns all verses in this surah
  • surahs numbers is between 1 and 114

https://quran-search-api.herokuapp.com/api/getVerse/{surah_id}/{verse_number}

Returns specific verse by taking

  • surah number
  • verse number

https://quran-search-api.herokuapp.com/api/getRangeOfVerses/{start}/{end}

Returns range of verses by taking:

  • start: where verses will start
  • end: where verses will end
  • verses in quran starts from 1 to 6236

https://quran-search-api.herokuapp.com/api/magicSearch/verseStartsWith/{letters}

takes one letter or more and then returns all verses that starts with this letters

https://quran-search-api.herokuapp.com/api/magicSearch/verseEndsWith/{letters}

takes one letter or more and then returns all verses that ends with this letters
